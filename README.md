# Tic-Tac-Toe Game

## Description
A console-based Tic-Tac-Toe game developed using Core Java for two players.

## Technologies Used
- Core Java
- Arrays
- Loops
- Scanner
- Conditional Statements

## Features
- Two-player game
- Win detection
- Draw detection
- Input validation
- Turn-based gameplay
